### Installation

```` 
 $ rvm use 2.0.0@kondocoin --create
 $ gem install rails
 $ bundle install --without production
 $ bundle update
 $ bundle install
````
If the app complains about a missing database.yml, please generate a
dummy rails environment and copy the default database.yml from it.
